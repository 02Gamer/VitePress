# 宝可梦朱紫

## 注意事项

道具打印机是DLC内容，游戏进度已捕获到`太乐巴戈斯`，并且道具打印机升级到`大师球级别`。

日期表格提供的`等待时间`，需游戏操作中提前1~2秒按A`我要打印！`，每个机子运行速度不同时间也不同，需自行测试。

## 打印机乱数操作

1. 在`我要打印！`界面按HOME键去改`稀有精灵球机会`或`道具双倍`的日期，返回游戏按A进入打印机界面，选择`打印次数1次`。

<!-- 1. 在`我要打印！`界面按HOME键去设置修改提供的日期，返回游戏按A进入界面，选择`打印次数1次`，触发`稀有精灵球机会开始了~！`或`道具双倍`前置条件。 -->

2. 触发**精灵球**或**道具双倍**后，在`我要打印！`界面，这里准备个`计时器`。选择想要物品根据提供的日期表格修改对应的日期时间，同时按下OK和启动计时器。返回游戏界面，`等待时间`到达按A进入，选择`打印次数&次`，这里的次数是日期表格对应的次数（1次/5次/10次）。
