# 宝可梦朱紫

**宝可梦 朱**（日文︰*ポケットモンスター スカーレット*，英文︰*Pokémon Scarlet*）和**宝可梦 紫**（日文︰*ポケットモンスター バイオレット*，英文︰*Pokémon Violet*）是两款《宝可梦》系列中第九世代的游戏。本作的冒险舞台是以现实世界的伊比利亚半岛为原型的帕底亚地区。

本作在宝可梦发表会 **2022.2.27** 中首次公开。

本作对应的游戏平台为**Nintendo Switch**。
